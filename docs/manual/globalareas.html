Global Areas

A GLOBAL AREA is an AREA that is not restricted to the MASK. Is is a rectangular AREA that can appear anywhere inside the main MASK, and is usually used to extract images from the design, like buttons, bullens, logos, etc.

We will use a GLOBAL AREA to cut the button images.

Select "Insert global area" from the left toolbar, and draw a box around the middle button. Adjust its size with the "Position and Size" tool.

Right-click on the GLOBAL AREA, and select properties.

Set the output parameters, such as the filename, and close the dialog window.

Now, when generating the HTML file, the area image will be saved to the VIEW image directory.
