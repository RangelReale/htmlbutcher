Position and Size

  • Positions:

    • Left, Top, Right, Bottom: Pixel positions of the LINEs that define the AREA

  • Movement:

    • Left/Top: Moves Left/Right or Top/Bottom LINEs simultaneously

  • Size:

    • Width/Height: Changes size, moving first right/bottom LINE if possible, else moving left/top LINE
