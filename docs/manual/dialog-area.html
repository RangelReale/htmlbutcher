Area

Properties

  • Name: Area name (optional)

  • Kind: Area kind:

    • None: Empty table cell

    • Image: Image inside table cell (or as cell background)

    • Mask: Sub-mask inside area (HTML table inside an outer table cell)

  • Background image: If checked, the image will be set as the cell's background, instead of a <IMG> tag

  • Layout type: The output layout type for this AREA's inner MASK - DEFAULT will use the VIEW setting. This can be different from the VIEW setting - if so, the generated page will use a mixed table/AP tableless layout (only available if Kind is MASK)

  • Image map: Area can contain an image map

  • Imap map name: Image map name

  • Base filename: Base filename for the generated image file (without extension)

  • Prepend parent area filename: If checked, the image file name will prepend recursivelly the parent areas filenames, and the view name, in the filename. if not checked, the image file name will be the base filename plus the file extension.

  • Use alternate file: Use the selected alternate image, instead of the default MASK image

  • Image formats: Image formats maintenance, with AREA image preview

HTML

  • Align: Cell ALIGN property

  • Vertical align: Cell VALIGN property

  • Background repeat: Background repetition option (only if background image)

  • Inner scrollable: If checked, puts a <DIV> inside the table cell

  • Cell tag append: <TD>/<DIV> tag smart attributes

  • Table tag append: Inner <TABLE> tag smart attributes (only if Kind is MASK)

  • Background color: Cell background color

  • Variable size: If checked, cell will not have a WIDTH attribute

  • Content: HTML inside the <TD> tag

Image

  • Image URL: URL for the image

  • Image URL target: Target attribute for the URL link

  • Image tag append: <IMG> tag smart attributes

  • Image source: Area image source:

    • Default: The default area image

    • Image link: The image from the link selected below

  • Image link: Image to use for the image source
