Image Format Properties

  • Selection type: Action to take when double-clicking on an the Image Format's image pixel

  • Image format: Image format selection. Will reflect in the preview area below

  • Preview image format: Second image format selection, used only for image format comparisson. Will reflect in the preview area below

  • Hover: Color below the mouse cursor. Informational only

  • Selected: Single-click on a pixel to set its color to the "Selected" item. Informational only

  • Transparent Colors: List of colors to make transparent on the image

  • Transparent Pixels: List of pixels to be made transparent on the image
