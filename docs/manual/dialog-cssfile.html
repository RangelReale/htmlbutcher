CSS Files

  • Add: Adds a new CSS file

  • Edit: Edits the selected CSS file

  • Remove: Removes the selected CSS file

  • Name: CSS file item name

  • CSS text: CSS file text

  • File path: CSS file path (relative from VIEW output path)

  • Save filename: CSS file output filename (with extension)
