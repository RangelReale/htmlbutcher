Image Formats

  • Add: Adds a new image format

  • Edit: Edits the selected image format

  • Remove: Removes the selected image format

  • Auto-update: When checked, auto-update preview image based on selection

  • Name: Image format name

  • Format: Image file format

  • JPEG compression: Compression for the JPEG format

  • JPEG progressive: JPEG format progressive flag

  • GIF/PNG8 QUantization: Bitdepth-reducing algorithm - Xialon-Wu is faster, NewQuant is better looking but slow (default is NewQuant)

  • Force bit depth if possible: Force output bit depth
