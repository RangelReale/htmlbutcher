Starting a project

To start a new project, use the File → New menu.

First you need to add one or more FILES. Go to the Data → Files menu, and add one or more layout files.

Figure 2.1. Files dialog


Next you need to create one or more MASKS. Go to the Data → Masks menu, and add one or more masks.

Figure 2.2. Masks dialog


Now you create a VIEW for each of the HTML pages that will be generated. Go to the Data → Views menu, and add one or more views.

Figure 2.3. View dialog


On the main screen, select the newly created VIEW on the "View" selection box.
