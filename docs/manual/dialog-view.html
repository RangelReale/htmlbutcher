View

View

  • Name: View name

  • Layout type: The output layout type for this view - DEFAULT will use the project options setting.

  • File: View image file

  • Mask: View mask

  • Image path: Path for this view's images

  • Filename: View filename (e.g.: index.html)

  • Encoding: Output file encoding

  • Items: Items to generate when saving this view:

    • HTML: Generate a HTML file

    • Images: Generate the area image files

    • Images Global: Generate the global area image files

  • Assorted file groups: Assorted files from the selected groups will be copied to the VIEW image output folder

  • CSS: Shortcut to edit the VIEW selected CSS file - if no CSS file is selected, does nothing

HTML

  • CSS file: CSS file to use in this VIEW - a CSS include will be generated on the HTML page

  • Background image: Image to be used on the body background (<BODY background="image">)

  • Background repeat: Background image repetition option

  • Background color: Body background color

  • Stretchable layout: Mark the view as generating a stretchable HTML page

  • Body tag add: Smart attributes to be added to the <BODY> tag

  • Title: HTML page title

  • Head: HTML inside the <HEAD> tag

  • Table tag add: Smart attributes to be added to the root <TABLE> tag

  • Body prepend: Text to append right after the <BODY> tag

  • Body append: Text to append right before the </BODY> tag

Alternate

  • Alternate file 1..8: Numbered alternate image files for the view
