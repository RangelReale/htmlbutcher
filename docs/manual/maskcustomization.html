Mask customization

A MASK can be reused in more than one VIEWS. In each VIEW, each AREA can have VIEW-specific customizations. Any area that do not have customizations, will have the same settings on all views.

To set an AREA to be customized, right-click on the AREA and select "Customize".

When an area is customized, its name appears in a bold font.

In a customized area, most AREA settings are specific to the current VIEW. Other VIEWs can have their own customizations, independent of each other.

The following settings are NOT customizable, they are the same on all VIEWS that use the same MASK:

  • Name

  • Position

  • Size

Using this feature, you can create a MASK for the whole site that slices the top title and left/right menus, and customize only the body area on all pages.

Caution

Any changes made to AREAs not customized, will reflect in all VIEWs using the same MASK.
