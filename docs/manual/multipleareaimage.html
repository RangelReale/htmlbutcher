Multiple images from one area

In HTMLButcher it is possible to generate more than one image from the same AREA, using alternate files.

To do this, right-click on the AREA, and select "Alternate files".

On this screen, you can enable one extra image per alternate file of the current VIEW. Clicking the "Enabled" checkbox for each file, will generate a new image file on the VIEW image directory. Each image can have its own properties, like format, filename, and transparency.
